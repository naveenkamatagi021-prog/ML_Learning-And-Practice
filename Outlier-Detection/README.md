## Project Overview
Outliers are extreme data points that are very different from the rest of the dataset. They can affect data analysis and reduce machine learning model performance if not handled properly.

In this practice, I explored different ways to detect and remove outliers using both visualization techniques and statistical methods.

## Methods Used

### 1. Boxplot
Boxplots help visualize the spread of data and easily highlight extreme values.

### 2. Histogram
Histograms show the distribution of data and help identify unusual data points.

### 3. Distribution Plot
Distribution plots help understand data shape and spot values that lie far from the normal range.

### 4. Z-Score Method
The Z-score method measures how far a data point is from the mean in terms of standard deviations.

### 5. IQR Method
The Interquartile Range (IQR) method detects outliers by using the lower and upper bounds of the data.
