# Handling Categorical Data Learning and Practice

This repository contains my hands-on practice and learning exercises on **Handling Categorical Data**, an important data preprocessing step in Machine Learning.

## About This Repository
This repository focuses on understanding how to work with categorical features, manage missing values, and prepare data for Machine Learning models through practical examples.

## Topics Covered
- Introduction to Categorical Data  
- Types of Categorical Variables  
- Handling Missing Values  
- Mode Imputation for Categorical Features  
- Forward Fill and Backward Fill Techniques  
- Data Cleaning and Preprocessing  
- Encoding Categorical Variables  
- Label Encoding  
- One Hot Encoding  
- Using Scikit-learn for Preprocessing  
- Data Visualization for Missing Values  
- Feature Preparation for Machine Learning  

## Purpose
The purpose of this repository is to:
- Strengthen data preprocessing skills  
- Practice handling categorical features  
- Understand encoding techniques  
- Build a strong foundation for Machine Learning workflows
