# One Hot Encoding Learning and Practice

This repository contains my hands-on practice and learning exercises on **One Hot Encoding**, an important feature engineering and data preprocessing technique in Machine Learning.

## About This Repository
This repository focuses on understanding and implementing One Hot Encoding to convert categorical data into numerical format suitable for Machine Learning models.

## Topics Covered
- Introduction to One Hot Encoding  
- Categorical Data Transformation  
- Nominal Feature Encoding  
- Dummy Variables Creation  
- Avoiding Dummy Variable Trap  
- Using Pandas for One Hot Encoding  
- One Hot Encoding with Scikit-learn  
- Feature Engineering Basics  
- Data Preprocessing Workflow  
- Preparing Data for Machine Learning Models 

## Purpose
The purpose of this repository is to:
- Strengthen feature engineering skills  
- Practice encoding categorical variables  
- Understand data preprocessing techniques  
- Build a strong foundation for Machine Learning workflows
