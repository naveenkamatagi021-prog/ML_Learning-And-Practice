🧠 Topics Covered
🔹 Feature Selection Techniques

Feature selection is used to choose the most relevant features for model building.

✅ Forward Selection
Begins with no features
Adds features one at a time
Selects features based on performance improvement
✅ Backward Elimination
Begins with all features
Removes least important features step by step

💡 Key Understanding:
Not all features contribute equally—removing unnecessary ones improves efficiency and accuracy.

🔹 Train-Test Split

Data splitting is performed using train_test_split from Scikit-learn.

📊 Understanding the Split
Training Data → Used to train the model
Testing Data → Used to evaluate performance

💡 Key Understanding:
Splitting data helps in testing the model on unseen data and avoids overfitting.
